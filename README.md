# streamlit_Apps
This repository contains multiple Streamlit applications. 
Use this index to navigate to each app.

## Available Apps
- **Bookkeeping**  
  A simple bookkeeping app for managing and tracking financial records.
- **InageDeskewer**  
  An image deskewing tool to correct the orientation of scanned or photographed documents.
- **EDA_explore**  
  An ollama driven tool for exploratory data anlysis on structure data like excel, csv.
- **transcription**  
  A local MVP meeting transcription app built with **Python** and **Streamlit**. Upload audio files, transcribe with [faster-whisper](https://github.com/SYSTRAN/faster-whisper), view timestamped segments, optionally convert Chinese output to Traditional Chinese, and export to TXT, SRT, or DOCX.
